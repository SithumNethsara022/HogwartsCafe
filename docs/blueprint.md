# **App Name**: Marauder's Menu

## Core Features:

- Sorting Hat Menu Tool: A generative AI tool that uses reasoning to suggest food items from the Hogwarts menu based on your mood, House preference, or cravings.
- Dynamic Magical Menu Engine: Full display of the categorised menu including Bites, Rice, and Pasta with logic to handle Half and Full portion price toggles.
- Cauldron Cart System: An interactive cart enabling users to add unlimited items, select quantities, and manage orders with real-time total price calculation.
- House Pride Theme Switcher: A custom UI feature that allows users to toggle the site theme between Gryffindor, Slytherin, Ravenclaw, and Hufflepuff colors.
- Hogwarts Special Customizer: A complex selection tool for special items like Nasi Goreng or Noodles, allowing choice of various toppings like prawns and cashew.
- Menu Schema Persistence: Backend implementation of the provided menu dataset using structured JSON storage for high-speed item fetching and search.
- Digital Receipt Store: Database implementation for logging historical orders and cart snapshots.

## Style Guidelines:

- Enchanting glassmorphism design featuring frosted glass blurred panels for the cart, menu headers, and detail cards to evoke a magical aesthetic.
- House themes: Gryffindor (Scarlet #740001, Gold #D3A625), Slytherin (Green #1A472A, Silver #AAAAAA), Ravenclaw (Blue #0E1A40, Bronze #946B2D), and Hufflepuff (Yellow #ECB939, Black #000000).
- Base background color: Deep Shadowed Parchment (#1e1b15) to create a dark, mystical environment for the frosted glass elements.
- A vintage pairing: 'Literata' (Serif) for headers to provide an ancient manuscript feel and 'Inter' (Sans-serif) for body text and cart controls.
- Custom magical icons such as Quills for order logs, Cauldrons for the cart, and Wands for settings with a metallic gradient finish.
- Smooth, ethereal hover states using glowing CSS filters and subtle floating vertical movements for highlighted menu items.
